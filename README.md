# FICHA_AOP

**FICHA_AOP** é um site interativo para criação e gerenciamento de fichas de personagens do universo *Ordem Paranormal*. Este projeto é **fanmade**, desenvolvido pela comunidade para fãs, com o objetivo de facilitar a criação de personagens e tornar as sessões mais dinâmicas e organizadas.

O site é totalmente responsivo e permite criar, atualizar e resetar fichas de forma prática e rápida, mantendo a experiência imersiva no universo do RPG.

---

## Sobre o Universo

*Ordem Paranormal* é um cenário de RPG moderno que mistura investigação sobrenatural, horror e elementos urbanos. Os jogadores assumem papéis de personagens que enfrentam ameaças paranormais enquanto investigam mistérios e desenvolvem suas habilidades. Este projeto foi criado para a comunidade, sem fins comerciais, com o objetivo de apoiar jogadores e mestres do jogo.

---

## Tecnologias Utilizadas

- **HTML5** – Estrutura do site  
- **CSS3** – Estilização e responsividade  
- **JavaScript** – Lógica e interatividade  
- **Git & GitHub Pages** – Controle de versão e hospedagem  

---

## Funcionalidades

- Criação dinâmica de fichas de personagens  
- Atualização em tempo real de atributos, perícias e indicadores  
- Sistema de reset para iniciar novas fichas  
- Interface limpa e responsiva, compatível com diferentes dispositivos
